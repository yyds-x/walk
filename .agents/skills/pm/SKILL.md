# PM Skill Bundle

## Routing

在执行前先判断用户意图，并阅读对应子技能的 `SKILL.md`：

- 数据分析/指标归因/漏斗留存：`pm-analytics/`
- 竞品拆解：`pm-competitor-deconstructor/`
- 实验设计/A-B：`pm-experiment-designer/`
- 截图复刻为 Pencil：`pm-image2pencil/`
- 截图复刻为 HTML 原型：`pm-image2proto/`
- 上线复盘/事故复盘：`pm-postmortem-writer/`
- PRD 需求文档：`pm-prd-writer/`
- 优先级排序（RICE/ICE/Kano）：`pm-prioritization-engine/`
- 多角色评审会：`pm-review-board/`
- 版本规划/路线图：`pm-roadmap-planner/`
- 调研问卷设计：`pm-survey-designer/`
- 埋点方案/指标口径：`pm-tracking-spec-writer/`
- URL 克隆为 Web 原型：`pm-url2proto/`

优先选择最贴近用户目标的子技能；若用户只说“帮我看看/整理一下”，先从 `pm-prd-writer` 或 `pm-review-board` 切入再细分。
