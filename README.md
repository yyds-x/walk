# Walk - WeChat Mini Program

这是一个基于微信小程序原生框架开发的项目，使用了 TypeScript 进行开发。

## 📋 项目简介

本项目包含了一个完整的底部导航栏（Tab Bar）实现，展示了“推荐”、“福利”、“订单”和“我的”四个主要功能模块。

## ✨ 主要功能

*   **自定义底部导航栏 (Custom TabBar)**:
    *   使用了微信小程序的 `custom-tab-bar` 组件能力。
    *   实现了自定义图标和选中态样式。
    *   解决了原生 TabBar 不支持 SVG 或复杂交互的问题。
    *   包含四个 Tab：推荐、福利、订单、我的。

*   **页面模块**:
    *   **推荐 (Recommend)**: `/pages/index/index`
    *   **福利 (Welfare)**: `/pages/welfare/welfare`
    *   **订单 (Order)**: `/pages/order/order`
    *   **我的 (My)**: `/pages/my/my`

## 🚀 快速开始

1.  **环境准备**: 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)。
2.  **导入项目**:
    *   打开微信开发者工具。
    *   选择“导入项目”。
    *   选择本项目根目录 (`/Users/allin/Desktop/walk` 或你存放的路径)。
    *   AppID 使用测试号或你自己的 AppID。
3.  **运行**:
    *   导入后，工具会自动编译并预览。
    *   确保开启了“允许不校验合法域名、web-view（业务域名）、TLS版本以及HTTPS证书”选项（如果在开发环境下）。

## 🛠 开发说明

*   **TypeScript**: 本项目使用 TypeScript，代码编译后会生成对应的 `.js` 文件。
*   **样式**: 使用 `.wxss`，支持 CSS 大部分特性。
*   **自定义 TabBar**: 修改 `miniprogram/custom-tab-bar/` 下的文件可调整底部导航栏样式和逻辑。修改 `miniprogram/app.json` 中的 `tabBar` 配置仅作为降级兼容和页面注册使用。

## 📝 备注

*   TabBar 图标资源位于 `miniprogram/images/tabbar/`。
*   若遇到 TabBar 不显示或样式异常，请检查 `custom-tab-bar/index.ts` 中的 `selected` 状态管理。
